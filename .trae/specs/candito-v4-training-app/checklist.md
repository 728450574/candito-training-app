- [ ] 数据刷新后不丢失（localStorage持久化）
- [ ] 空状态处理（无周期时引导创建）

### 核心逻辑
- [ ] 6周计划百分比计算正确（与Excel对照）
- [ ] 第1周为5个训练日（非4个）
- [ ] 第1周Day3=上肢（非休息）

- [ ] 第5周完成 → 预估1RM系数正确（2次×1.03, 3次×1.06, 4次×1.09），实时计算不持久化

- [ ] 数据层（services/stores）与UI层（views/components）无直接耦合
- [ ] 所有 localStorage key 使用指定命名（candito_cycles, candito_records_{id}, candito_metrics, candito_settings, candito_active_cycle）
